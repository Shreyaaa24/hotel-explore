# 🏨 Hotel Explorer

A modern and responsive hotel browsing application built with **React** and **Vite**. This project integrates with a live Hotel API to display hotel listings and allows users to search, filter, and view detailed information about each hotel.

## 🚀 Features

* 📋 View a list of hotels from a live API
* 🔍 Search hotels by name
* 📍 Filter hotels by location
* 🏨 View detailed information for each hotel
* ⭐ Display hotel ratings
* 💰 Display hotel prices
* 📱 Fully responsive design
* ⚡ Fast performance using Vite


## 🛠️ Tech Stack

* React.js
* Vite
* React Router DOM
* JavaScript (ES6+)
* HTML5
* CSS3
* REST API
* Git & GitHub

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Shreyaaa24/hotel-explore.git
```

Navigate to the project directory:

```bash
cd hotel-explore
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the application in your browser:

```
http://localhost:5173
```
